# Mathew Monay Portfolio Website

## Overview
A personal portfolio website built with pure HTML, CSS, and JavaScript. Features a welcome overlay, scroll reveal animations, responsive mobile navigation, and multiple pages (Home, Projects, Contact).

## Project Structure
- `index.html` - Home page with hero section and featured projects
- `projects.html` - Projects showcase page
- `contact.html` - Contact information page
- `images.html` - Images gallery page
- `styles.css` - Global styles
- `main.js` - JavaScript for animations and interactions
- `server.js` - Simple Node.js static file server for development

## Running the Project
The website runs on a Node.js static file server on port 5000. The workflow "Portfolio Website" starts the server automatically.

## Deployment
Configured as a static site deployment, serving files from the root directory.
